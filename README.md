# ✨ 奇妙互动世界

一个充满趣味和互动的网页实验！🎨

## 🌟 功能特性

### ✅ 已实现的所有功能

1. **👻 小幽灵鼠标** - 鼠标变成小幽灵，跟随你的移动
2. **🌧️ 奇怪的雨** - 屏幕随机落下表情包、糖果、星星、小章鱼
3. **💫 点击特效** - 点击屏幕产生泡泡 + 撒星光 + 音效
4. **💃 跳舞文字** - 输入的文字会跳舞、躲鼠标、弹跳
5. **🐱 屏幕宠物** - 会发呆、求摸、躲光标、打哈欠的小宠物（双击有惊喜）
6. **🗑️ 情绪垃圾桶** - 输入烦恼一键烧掉
7. **🎲 真心话生成器** - 点一下生成离谱问题
8. **🌅 时间背景** - 背景随时间自动变色（早/中/晚/深夜）
9. **🌌 反重力** - 拖动方块会飘在空中弹来弹去
10. **🎉 隐藏彩蛋** - 按 `C` + `T` + `L` 解锁秘密界面

## 🎮 玩法说明

- **移动鼠标** - 小幽灵会跟着你
- **点击任意位置** - 产生泡泡和星光
- **抚摸宠物** - 点击屏幕宠物
- **双击宠物** - 看它打哈欠
- **拖动方块** - 体验反重力效果
- **按 CTL** - 解锁隐藏彩蛋！

## 🛠️ 技术栈

- HTML5
- CSS3（动画、渐变、响应式）
- JavaScript（原生，无框架）
- Web Audio API（音效）

## 🎨 自定义

### 修改宠物
编辑 `script.js` 中的 `petEmojis` 数组：
```javascript
const petEmojis = ['🐱', '🐶', '🐰', '🦊', '🐼'];
```

### 修改下雨内容
编辑 `script.js` 中的 `rainEmojis` 数组：
```javascript
const rainEmojis = ['🍬', '⭐', '🐙', '🎈', '🌸', '💖', '🦄', '🍭', '✨', '🎵'];
```

### 修改真心话问题
编辑 `script.js` 中的 `truthQuestions` 数组。
